# Personal Portfolio

A responsive portfolio website built with React and Vite to showcase my projects, technical skills, education, and professional experience.

## Features

* Responsive design
* Framer Motion animations
* Project showcase
* Resume download
* EmailJS contact form

## Tech Stack

* React
* Vite
* CSS3
* Framer Motion
* EmailJS

## Installation

```bash
npm install
npm run dev
```
